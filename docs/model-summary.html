CS175 Research Site

Model summary

The formal setup for a confidence-aware learning-augmented online scheduler.

Overview Model Milestones Reproducibility Results

Core entities

Job
Arrival time, true processing time, predicted duration, confidence, deadline, penalty.
MachineState
Current machine availability and queued assignments for identical parallel machines.
ExpertPolicy
One scheduling rule, such as EDF, SPT, or a hybrid confidence-gated rule.
MetaScheduler
Weights or switches among experts based on observed outcomes and confidence.

Default assumptions

  • Online arrivals over time
  • Non-preemptive execution
  • No migration between machines
  • Soft deadline penalties rather than hard feasibility
  • Predicted durations plus a scalar confidence score

Contribution shape

The intended novelty is a confidence-aware meta-policy that uses a prediction-aware expert while retaining a worst-case fallback against classical scheduling rules.
